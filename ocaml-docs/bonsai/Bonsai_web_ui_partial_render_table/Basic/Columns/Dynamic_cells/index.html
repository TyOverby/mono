Module Columns.Dynamic_cells

Dynamic_cells is a column-specification format with the following tradeoffs:

  • Pro: Each cell is it's own bonsai computation, so you can stick complex components in side of them, like forms, or graphs.
  • Con: The set of columns must be statically known ahead of time, and can not be determined dynamically.
