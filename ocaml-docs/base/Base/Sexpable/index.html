Module Base.Sexpable

Provides functors for making modules sexpable when you want the sexp representation of one type to be the same as that for some other isomorphic type.

module Of_sexpable (Sexpable : Sexplib0.Sexpable.S) (M : sig ... end) : Sexplib0.Sexpable.S with type t := M.t
module Of_sexpable1 (Sexpable : Sexplib0.Sexpable.S1) (M : sig ... end) : Sexplib0.Sexpable.S1 with type 'a t := 'a M.t
module Of_sexpable2 (Sexpable : Sexplib0.Sexpable.S2) (M : sig ... end) : Sexplib0.Sexpable.S2 with type ('a, 'b) t := ( 'a, 'b ) M.t
module Of_sexpable3 (Sexpable : Sexplib0.Sexpable.S3) (M : sig ... end) : Sexplib0.Sexpable.S3 with type ('a, 'b, 'c) t := ( 'a, 'b, 'c ) M.t
module Of_stringable (M : Stringable.S) : Sexplib0.Sexpable.S with type t := M.t

New code should use the [@@deriving sexp] syntax directly. These module types (S, S1, S2, and S3) are exported for backwards compatibility only.
